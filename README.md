# Amazon Clone - HTML & CSS
 # Project Overview

This is a front-end clone of the Amazon homepage built purely using HTML and CSS. The goal of this project is to practice web design, layout structuring, and responsive styling, while replicating the look and feel of a real e-commerce website.

The project is inspired by a tutorial video (YouTube Reference
) and is intended for learning purposes only. It does not include backend functionality like a shopping cart or user login.

# Features

 Fully structured Amazon-like homepage layout.

Hero section with banner images.

Product listing cards with images, names, and prices.

Navigation bar with search bar and menu links.

Footer with links and information sections.

Responsive design for different screen sizes.
